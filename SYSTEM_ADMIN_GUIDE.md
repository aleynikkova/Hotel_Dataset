# Руководство для системного администратора

## Обзор

Системный администратор имеет полный доступ к управлению системой бронирования отелей. Он может управлять отелями и пользователями.

## Вход в систему

1. Перейдите на главную страницу приложения
2. Нажмите "Войти" в правом верхнем углу
3. Введите email и пароль системного администратора
4. После входа вы будете автоматически перенаправлены на панель системного администратора

## Панель управления

После входа вы увидите две основные секции:

### 1. Отели
Управление всеми отелями в системе.

### 2. Роли пользователей
Управление пользователями и их ролями.

---

## Управление отелями

### Просмотр отелей
- Все отели отображаются в виде карточек
- Показывается: название, адрес, город, страна, телефон, email, статус, количество звезд
- Используйте фильтры для поиска:
  - Поиск по названию или городу
  - Фильтр по статусу (активные/неактивные)

### Добавление нового отеля

1. Нажмите кнопку **"Добавить отель"**
2. Заполните обязательные поля:
   - Название отеля
   - Город
   - Страна
   - Адрес
3. Опционально заполните:
   - Количество звезд (1-5)
   - Телефон
   - Email
   - Описание
   - Администратор отеля (можно назначить позже)
4. Установите статус "Отель активен"
5. Нажмите **"Сохранить"**

### Редактирование отеля

1. Найдите нужный отель в списке
2. Нажмите кнопку **"Редактировать"**
3. Измените необходимые поля
4. Можете назначить или изменить администратора отеля
5. Можете деактивировать отель (снять галочку "Отель активен")
6. Нажмите **"Сохранить"**

### Удаление отеля

1. Найдите нужный отель в списке
2. Нажмите кнопку **"Удалить"**
3. Подтвердите удаление

⚠️ **Внимание:** При удалении отеля будут удалены все связанные данные (номера, бронирования).

## Управление пользователями

### Просмотр пользователей

- Все пользователи отображаются в таблице
- Показывается: имя, email, телефон, роль, привязанный отель (для админов отелей), статус
- Статистика сверху показывает общее количество пользователей по ролям

### Фильтрация пользователей

Используйте фильтры для поиска:
- Поиск по имени или email
- Фильтр по роли:
  - Гости
  - Администраторы отелей
  - Системные администраторы
- Фильтр по статусу (активные/заблокированные)

### Изменение роли пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"Изменить"**
3. Выберите новую роль:
   - **Гость** - обычный пользователь, может бронировать номера
   - **Администратор отеля** - управляет конкретным отелем (номерами, бронированиями)
   - **Системный администратор** - полный доступ к системе

4. Если выбрана роль **"Администратор отеля"**:
   - Обязательно выберите отель из списка
   - Пользователь получит доступ только к этому отелю

5. Нажмите **"Сохранить"**

### Блокировка пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"Изменить"**
3. Снимите галочку **"Пользователь активен"**
4. Нажмите **"Сохранить"**

Заблокированный пользователь не сможет войти в систему.

### Разблокировка пользователя

1. Найдите заблокированного пользователя (используйте фильтр "Заблокированные")
2. Нажмите кнопку **"Изменить"**
3. Установите галочку **"Пользователь активен"**
4. Нажмите **"Сохранить"**

## Важные заметки

### Роли пользователей

1. **Гость (guest)**
   - Регистрируется самостоятельно
   - Может искать отели, бронировать номера
   - Может просматривать свои бронирования
   - Может оставлять отзывы

2. **Администратор отеля (hotel_admin)**
   - Назначается системным администратором
   - Привязан к конкретному отелю
   - Может управлять номерами своего отеля
   - Может управлять бронированиями своего отеля
   - Может просматривать информацию об отеле

3. **Системный администратор (system_admin)**
   - Полный доступ ко всей системе
   - Может добавлять/редактировать/удалять отели
   - Может управлять ролями пользователей
   - Может назначать администраторов отелей

### Процесс назначения администратора отеля

1. Пользователь регистрируется в системе как гость
2. Системный администратор находит этого пользователя в списке
3. Меняет роль на "Администратор отеля"
4. Выбирает отель, которым будет управлять пользователь
5. Сохраняет изменения
6. При следующем входе пользователь получит доступ к панели управления отелем

### Администратор отеля в настройках отеля

- При создании отеля необязательно сразу назначать администратора
- Можно назначить администратора позже при редактировании отеля
- В выпадающем списке доступны только пользователи с ролью "Администратор отеля"
- Один администратор может управлять только одним отелем
